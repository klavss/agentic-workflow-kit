# Framework Policy

A reusable, mode-aware agentic workflow for cross-project development.

## Modes
- Explore
- Build
- Harden
- Release

## Base flow
spec-to-plan -> checkpoint-review -> approval -> milestone-autopilot -> drift-audit

## Conditional controls
initiative-index, milestone-scorecard, security-baseline, dependency-governance, release-gate, framework-governance.

## Confidence contract
Major outputs include: confidence, evidence basis, unknowns, next probe/check.

## Trivial-task bypass
Only when single-step, low-risk, no architecture/security/dependency/API/schema impact, and one check verifies completion.
